BackScreen 설치 안내 (비개발자용)

1) 설치 파일 받기
- 전달받은 "BackScreen_Setup_vX.Y.Z.exe" 파일을 다운로드합니다.

2) 설치 진행
- exe 파일을 더블클릭합니다.
- Windows 경고가 나오면 "추가 정보" -> "실행"을 클릭합니다.
- 설치 완료 후 바탕화면/시작 메뉴의 BackScreen을 실행합니다.

3) 처음 설정 (약 10분)
- 화면 상단 체크리스트 순서대로 입력합니다.
  1. 학교 정보 입력
  2. 이번 달 급식 등록
  3. 이번 달 일정 등록
  4. "저장 + 배경 적용" 클릭

4) 매달 업데이트
- "이번 달 빠른 수정"에서 일정/급식만 수정합니다.
- 마지막에 "저장 + 배경 적용"을 누릅니다.

5) 백업/복구
- 백업: 앱에서 "JSON 내보내기" 클릭
- 복구: 앱에서 "JSON 가져오기" 선택

6) 앱 제거(완전 삭제)
- 앱 화면 오른쪽의 "앱 제거" 패널에서 확인 입력칸에 정확히 "삭제"를 입력합니다.
- "앱 제거 실행"을 누르면 언인스톨러가 바로 실행됩니다.
- 제거 시 앱 데이터(app.db/backups/wallpaper)는 완전 삭제됩니다.
- 제거 전에 기존 바탕화면으로 원복을 시도합니다.
- 앱 내 제거가 실패하면 Windows "설정 > 앱 > 설치된 앱"에서 BackScreen을 제거합니다.

문제가 있을 때
- 앱이 열려 있으면 다시 실행하지 말고, 현재 창에서 "저장 + 배경 적용"을 한 번 더 눌러보세요.
- 설치 파일 버전이 최신인지 확인하세요.
