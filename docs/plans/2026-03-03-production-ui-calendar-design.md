# BackScreen 프로덕션 UI/UX + 학교 교육 활동 캘린더 디자인

- 작성일: 2026-03-03
- 범위: 편집 앱(App)과 바탕화면 렌더(renderWallpaper) 동시 개선

## 승인된 결정

1. 범위: 편집 앱 + 바탕화면 결과물 모두 개선
2. 학교 교육 활동: 월간 캘린더 그리드 + 하단 일정 카드
3. 비주얼: 일러스트 강조 톤 + 귀여운 장식 요소
4. 최소 지원 해상도: 1366x768
5. 편집 앱 구조: 2열 + 상단 핵심 버튼 + 우측 미리보기 카드
6. 텍스트 정책: 자동 축소 후 말줄임

## 목표

1. 비개발자도 즉시 이해되는 편집 흐름 제공
2. 바탕화면 결과물이 레퍼런스를 넘는 프로덕션 퀄리티 확보
3. 텍스트 깨짐/오버플로우 없는 안정 렌더링

## 정보 구조

### 편집 앱

- 상단: 핵심 실행바(저장/저장+적용/지금 적용 + 가져오기/내보내기)
- 본문 좌측: 월간 빠른 수정, 급식/행사 편집, 시간표/테마
- 본문 우측: 실시간 미리보기, 체크리스트 진행상황, 학교정보/앱관리

### 바탕화면

- 좌 12%/상 8% 안전영역 유지
- 헤더: 날짜/시간, 학교정보, 현재수업
- 본문 3열: 시간표, 급식, 학교 교육 활동(월간 캘린더 + 일정 카드)

## 렌더링 규칙

1. 모든 텍스트는 fitText(자동축소 -> 말줄임) 경유
2. 패널별 표시 개수 제한으로 세로 오버플로우 방지
3. 캘린더 셀은 날짜 숫자 중심, 이벤트는 점/강조표시
4. 하단 일정 카드는 월간 핵심 항목만 표시, 초과는 +N 요약

## 테스트 기준

1. 1366x768에서 PNG 생성 및 예외 없음
2. 긴 문자열에서도 카드 경계 이탈 없음
3. 캘린더 렌더 경로가 월간 일정 데이터와 연결됨
4. 편집 앱은 기존 핵심 플로우(저장/적용/언인스톨) 회귀 없음
